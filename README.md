

## What is used

- Latest Next.js 13 features
- Next.js App Router
- Next.js Server Actions
- Client & Server Components
- TypeScript 
- Tailwind CSS
- Context API
- Framer Motion
- React.Email & Resend
- Custom React hooks
- Light & Dark mode
- Responsive website

## Setup

1. Add RESEND_API_KEY environment variable in .env.local
2. In the send-email.ts action file, change the "to" email to your own email
